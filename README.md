# Requirements

- 3 different levels:
  - Easy: 4x4 board
  - Medium: 4x6 board
  - Difficult: 5x6 board
- Movement counter.
- Pairs found counter.
- Possibility to return to the main page.
- Extra: Record tracking for each level and possibility to change game duration.
